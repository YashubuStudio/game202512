
# N100.md

## AI計算ノード（戦略生成専用）

### 概要

N100は **思考専門ノード**。
ゲーム状態を「見る」だけで、**決して状態を確定しない**。

---

## 主な責務

### 1. 指令セット生成

* 現在状態に基づく最適指令生成
* 難易度別AI（easy/normal/hard など）
* 複数戦略の併存（攻撃型・経済型）

### 2. 時間制約付き思考

* `time_budget_ms` を厳守
* 途中結果でのフォールバック

### 3. 決定性補助

* `rng_salt` による多様性提供
* **seedは直接管理しない**

---

## 提供API（RPi5からのみ）

### `POST /think`

**入力**

```json
{
  "match_id": "m1",
  "turn": 12,
  "side": "A",
  "masked_state": {...},
  "time_budget_ms": 200,
  "nonce": "abc"
}
```

**出力**

```json
{
  "command": {...},
  "meta": { "score": 0.82 }
}
```

---

## 非責務

* 状態更新
* ルール解釈の最終決定
* 勝敗判定
* ログ保存

---
